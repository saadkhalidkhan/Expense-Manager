# Screenshots & demo assets

Add media here so they appear in the root [README.md](../../README.md).

## Recommended files

| File | Description |
| --- | --- |
| `dashboard.png` | Main dashboard (balance + recent transactions) |
| `add_transaction.png` | Add / edit transaction screen |
| `transaction_details.png` | Transaction detail view |
| `demo.gif` | Short screen recording (15–30 seconds) |

## Tips

- **Screenshots**: Use an emulator or device at ~1080×2400, PNG format.
- **GIF**: Record with [Android Studio Screen Record](https://developer.android.com/studio/debug/am-video), [scrcpy](https://github.com/Genymobile/scrcpy), or convert MP4 → GIF with [ffmpeg](https://ffmpeg.org/) or an online tool.
- Keep file sizes reasonable (< 2 MB per image, < 5 MB for GIF) for fast README loading on GitHub.

After adding files, commit and push — GitHub will render them automatically in the README.
