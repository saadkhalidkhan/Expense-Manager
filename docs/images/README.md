# Screenshots & demo assets

Add media here so they appear in the root [README.md](../../README.md).

## Recommended files

| File | Description |
| --- | --- |
| `dashboard.png` | Main dashboard (balance + recent transactions) |
| `add_transaction.png` | Add / edit transaction screen |
| `transaction_details.png` | Transaction detail view |
| `demo.webm` or `demo.gif` | Short screen recording (15–30 seconds) |

## Tips

- **Screenshots**: Use an emulator or device at ~1080×2400, PNG format.
- **Video**: Record with [Android Studio Screen Record](https://developer.android.com/studio/debug/am-video) or [scrcpy](https://github.com/Genymobile/scrcpy). Save as `demo.webm` (or convert to `demo.gif` with [ffmpeg](https://ffmpeg.org/)).
- Keep file sizes reasonable (< 2 MB per image; prefer a short, compressed demo under ~10 MB for GitHub).

After adding files, commit and push — GitHub will render them automatically in the README.
